# Orange_4_IT-homework-

Basic CRUD API tests for [galaxy.scalar.com](https://galaxy.scalar.com) using Playwright and TypeScript.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file with your credentials:
```
BASE_URL=https://galaxy.scalar.com
EMAIL=your-email@example.com
PASSWORD=your-password
```

## Running Tests

```bash
npm test
```
